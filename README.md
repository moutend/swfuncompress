# swfuncompress

Uncompress `.swf` (Macromedia Flash Data) file.

## Install

Download the latest golang and then run the command below:

```console
go install github.com/moutend/swfuncompress/cmd/swfuncompress@latest
```

## Usage

```console
swfuncompress -i foo.swf -o bar.swf
```
